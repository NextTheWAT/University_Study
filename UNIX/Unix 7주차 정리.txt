UNIX 7주차 정리

ls -l aaa[1-9].txt - aaa1~9파일 찾기
ls -s aaa*.txt - aaa이름이 들어간 파일들 찾기
cd ~ - 맨위 디렉토리
cd -  - 이전 작업 디렉토리


셸의 종류와 변경
- 셸 ?
   - 사용자와 커널 사이의 중간 역할
   - 사용자가 입력한 명령을 처리하고 실행 결과를 알려줌
   - 편리한 사용을 위해 다양한 기능 제공
- 셸의 종류
   - 본 셸(sh), 콘 셸(ksh), C 셸(csh), 배시 셸(bash) 등
   - 종류 확인하기
     - 프롬프트로 확인 : C 셸은 % / 본 셸, 콘 셸, 배시 셸은 $ 
     - 명령으로 확인
- 로그인 셸과 서브셸 ?
   - 로그인 셸 : 사용자가 로그인 직후 자동 생성되는 셸
   - 서브셸 : 사용자가 직접 실행한 셸

문자열의 출력
- echo
   - 셸 명령과 유틸리티 모두 제공
   - 지정한 문자열 출력
- print

셸 특수 문자
   - 셸 명령
   - 옵션이 제공되어 echo보다 편리
- * 특수 문자
   - 가장 자주 사용되는 특수 문자 → 임의의 문자열을 나타냄
- ?와 [ ] 특수 문자
   - 모두 하나의 문자를 나타내는 데 사용
   - ?는 임의의 한 문자, [ ] 괄호 안에 포함된 문자 중 하나
   - [ ] 다른 특수 문자와도 혼합해 사용할 수도 있다.
- ~와 – 특수 문자
   - 디렉토리를 나타내는 특수 문자
   - ~가 단독 사용 → 사용자의 홈 디렉토리
   - -는 cd 명령으로 디렉토리를 이전하기 직전의 작업 디렉토리
