뜻 외워두기@@  중간고사에 나옴 5주차 실습문제 참고
vi 편집기 --> UNIX에서 가장 많이 쓰는 편집@@@
[입력모드] <--> [명령모드] 
a,i,o,O	  	[esc]키
i = 그 자리에서 입력 @@ 
a = 다음 자리에서 입력 @@
o = 그 다음 행(라인)으로 입력 @@
O = 위의 행(라인)에서 입력 @@
r = 한 글자만 변경(겹쳐쓰기)
R = 계속 겹쳐쓰기
gg = 파일의 첫 행으로 이동
G = 파일의 마지막 행으로 이동

/단어 = 단어 찾기 @@
	n을 누르면 똑같은 다음 단어 찾기
/%s/단어/단어2/g = 단어 -> 단어2로 전체 바꾸기  @@

:n = n행으로 가기

k = 위로 가기
l = 오른쪽으로 가기
h = 왼쪽으로 가기
j = 아래로 가기

[지우기]
x = 원하는 글자에 가서 한 글자 지우기 @@
cw = 한 단어 변경하기
dd= 한 라인(행) 지우기
3dd = 3줄 지우기
쉬프트 + d = 한 줄 지우기
D = 줄 지우기

[복구]
입력후 esc 키를 누르고 u를 누르면 복구 @@

[복사]
yy : 1줄 복사, p 누르면 그 다음줄에 붙여넣기
yy p
2yy : 2줄 복사, p 누르면 그 다음줄에 붙여넣기
2yy p
3yy : 2줄 복사, P 누르면 위에 붙여넣기
nyy하면 n줄만큼 복사가 된다

[저장]
:w = 명령모드에서 - 저장
:w ~.txt = 다른이름으로 저장
:q! = 저장하지않고 종료
:w! = 저장후 편집종료

[시작, 종료]
:q = 끝내기
:wq = 저장하면서 끝내기 @@
:set number = 줄(행) 번호 부여
:set nonumber = 줄(행) 번호 제거
