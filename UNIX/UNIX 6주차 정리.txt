모드형과 비모드형

모드형
- 명령모드에서 입력한 키는 명령으로 간주
- 입력모드에서 입력한 키는 데이터로 간주
- 모드를 바꾸기 위한 특수 키가 있음 (예: Esc)
- vim은 모드형

비모드형
- 입력한 모든 키는 데이터로 간주됨
- 명령은 특수키(Ctrl, Alt 등)와 일반키의 조합으로 구성
- 예 : 아래 한글, 워드

vim
- 새로운 파일 시작
- 파일 저장할 때 이름 지정
- 유닉스 초기에 사용 → 지금은 거의 사용하지 않음

vim 파일이름 - 이름 쓰는걸 추천
- 지정한 이름이 없으면 새로운 파일 생성
- 지정한 이름이 있으면 기존 파일 열기
- vim first → first라는 파일이 열림
- vim → 빈 파일이 열림

:q - vim에서 작업한 것이 없을 때 그냥 종료한다
:q! - 작업한 내용을 저장하지 않고 종료한다.
:w [파일명] - 작업한 내용을 저장만 한다. 파일명을 지정하면 새 파일로 저장한다.
:wq :wq! - 작업한 내용을 저장하고 vim을 종료한다.
ZZ(쉬프트+zz - 작업한 내용을 저장하고 vim을 종료한다. - 교수님 한번도 안써봄 시험x

입력 모드로의 전환  @@
I, A 는 안나옴

i - 커서 앞에 입력한다(현재 커서 자리에 입력).
a - 커서 뒤에 입력한다(현재 커서 다음 자리에 입력).
o - 커서가 위치한 행의 다음 행에 입력한다
O - 커서가 위치한 행의 이전 행에 입력한다.

편집 기능
- 복사하기 / 붙이기 / 잘라내기 명령
- 버퍼의 사용
- 마지막 행 모드에서 복사하기와 잘라내기
- 검색하기 / 바꾸기

편집 기능 @
복사하기 / 붙이기 / 잘라내기 명령
yy - 커서가 위치한 행을 복사한다
p - 커서가 위치한 행의 아래쪽에 붙인다.
P - 커서의 위치한 행의 위쪽에 붙인다.
dd, #dd - 커서가 위치한 행을 잘라둔다(#에는 잘라둘 행의 수 지정, 예를 들어
      3dd 명령을 입력하면 3행을 잘라 두기(삭제)).

검색하기 / 바꾸기 @
- 바꾸기
   - 기존의 문자열을 다른 문자열로 바꾸려면 먼저 :를 입력
       → 마지막 행 모드로 이동
   - 파일 전체 대상 또는 특정 범위만 지정 실행 가능
/문자열 - 문자열을 아래 방향으로 검색한다 
n - 원래 찾는 방향으로 다음 문자열을 찾는다.
N - 역방향으로 다음 문자열을 찾는다.
%s/문자열1/문자열2/g - 파일 전체에서 모든 문자열1을 문자열2로 바꾼다.

vi 환경을 설정하는 특수 명령과 변수들 @
:set nu - 파일 내용의 각 행에 행 번호를 표시한다
:set nonu - 행 번호를 감춘다
