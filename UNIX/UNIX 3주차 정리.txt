 d 디렉토리 파일
 l 심볼릭 링크 파일
 b 블록 단위로 읽고 쓰는 블록 장치 특수 파일
 c 섹터 단위로 읽고 쓰는 문자 장치 특수 파일
 p 파이프, 프로세스간 통신에 사용되는 특수 파일
 s 소켓, 네트워크 통신에 사용되는 특수 파일

일반 파일, 디렉토리 파일, 심볼릭 링크 파일, 장치 파일

일반 파일
 - 데이터의 저장을 목적으로 하는 파일
 - 텍스트 파일
 	- 파일의 내용이 아스키(ASCII)코드로 구성된 파일
 	- 텍스트 편집기나 cat, more 명령을 이용하여 그 내용을 볼 수 있다.
 - 바이너리 파일 : 우분투로 보면 파일이 깨져보임 (c언어 등)
 	- 내용이 아스키 코드가 아닌 모든 파일
	 - 지정된 응용(실행) 프로그램을 이용하여야 내용을 볼 수 있다.
 - 디렉토리 파일 : 앞에 d 가 붙음(폴더도 포함)
 	- 내용이 다른 파일이나 하위 
   	  디렉토리의 이름인 특수 파일
 - 심볼릭 링크 파일(바로가기)
 	- 원본 파일을 가리키는 특수 파일텍스트 파일
 - 장치 파일(마우스,키보드,프린터)
	 - 장치를 관리하기 위한 특수 파일
	 - 데이터를 저장하기 위한 데이터 
    	    블록이 없음
	 - 파일 크기 대신 장치를 구분하기 위한
  	    두 개의 숫자 저장
