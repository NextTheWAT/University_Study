웹의 동작
요청 : 클라이언트에서 서버로 정보를 요구하기 위해 보내는 메시지입니다. 
	 이 요청 방식에는 GET방식과 POST 방식이 있습니다.
응답 : HTTP에서 요구된 메시지에 대한 응답, HTML, 이미지 등이	
 	 응답의 내용이 됩니다.


과제 : GET, POST 방식이 무엇인지 출력해서 제출

GET, POST 잘 알아두기


정적 페이지, 동적 페이지

정적 페이지 : 변하지 않는 성격을 가진 페이지
동적 페이지 : 자동으로 생성된 페이지, 기존의 웹 서버에새로운 기능을 
	           가진 프로그램을 탑재하여 동적으로 HTML을 생성


 CGI(Common Gateway Interface)
 동적 페이지를 위한 기술
 웹 서버와 동적 컨텐츠 생성을 맡은 프로그램 사이에서 정보를 주고받는 
  인터페이스
 초기 웹 프로그래밍에 사용된 기술
 프로세스 단위로 실행되기 때문에 사용자 증가 시 급격한 성능 저하


확장 CGIrvlet, JSP 등

JSP - Java Server Page


JSP란 무엇인가?
 웹 프로그래밍 언어들 중의 하나
 동적인 페이지를
 전통적인 CGI 방식의 단점 보완
 프로세스 생성방식의 변화로 시스템의 부하를 줄임
 ASP, PHP (c언어 기반), Se 생성하기 위한 서버측 스크립트 언어
 자바언어를 기반
 객체지향적
 플랫폼 독립적 - 어느 운영체제(OS)에서든 실행 가능 자바는 JVM으로 실행되기 때문에
 네트워크 지향적
 뛰어난 보안성
 멀티스레드 기능
 친근한 코드
