2. IIS 웹 서버 구축

웹 서버 루트 디렉토리의 위치
C:\inetpub\wwwroot\



코드 : cd [MYSQL이 다운된 경로]   => 그 경로로 이동됨
코드 : mysqld --install      => 저는 아래와같이 이미 존재한다고(already exists!)나오지만, 아마 Service successfully installed라고 뜨실거임
코드 : Mysqld --initialize
