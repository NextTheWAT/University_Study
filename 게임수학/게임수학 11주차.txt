static 변수?
- 게임에서 매우 중요한 기능
- 외부에서 바로 액세스가 가능한 변수
- 씬이 전환되더라도 계속 값이 남아 있는 변수

사용 방법
- 선언
- public static <변수 형> <변수 명>	@
- 액세스 방법
< 클래스 이름>.<변수 이름>	@

get, set 사용하는방법도 잘 알아두기
