UX(User eXperience Sloution) Design
사용자 경험을 디자인한다.
사용자 경험 문제해결


디자인과 UX 디자인

디자인
- 디자인은 어떤 객체, 시스템, 활동, 프로세스 등을 구현하기 위한 계획이나 만들려는 사양 또는
(프로토타입, 제품, 프로세스 형태로 된) 결과물을 의미한다.


UX디자인
- UX디자인은 경험 자체를 새롭게 디자인하겠다는 의미


UX(UX, user experience, 사용자 경험)
- 사용자가 디지털 제품이나 서비스 사용에서 갖게 되는 모든 지식과 기억과 감정
	사용자가 디지털 제품이나 서비스를 사용하기 전, 중, 후에 일어나는 감정, 신념, 선호도, 지각, 신체적
	정신적 반응이나 행동을 포함하는 매우 넓은 개념


UX의 특징
- 주관성 : 똑같은 디지털 시스템에 대해
		각자 다른 지식과 기억과 감정에 의한 모두 다른 경험
- 총체성 : 특정 시점에 특정 개인이 느끼는 총체적인 경험이기 때문에
		구제적인 요소들로 구분하기 어렵다
- 정황성 : 상호작용이 일어나는 시점에서의 맥락에 영향을 받는다.
=> UX의 세가지 특징에 대해 자신의 경험을 통해 설명해 주세요. (택1)


시험은 주관식이다@


HCI의 목표
- 사용자에게 최적의 경험(UX, FLOW)를 제공하고자 한다.
최적의 경험을 제공하기 위한 최소한의 조건
- 사용성 : 목적을 달성하고자 하는 과정이 효율적이어야 한다.
- 유용성 : 하고자 하는 일을 달성할 수 있어야 한다.
- 감성 : 적절한 느낌을 받을 수 있어야 한다.
- 시스템의 성격에 따라 더, 혹은 덜 중요할 수는 있다.
	조화가 중요, 때로는 상호 대치하는 관계가 되기도 한다.


UI(user interface, 사용자 인터페이스)
- 사용자가 접촉하는 디지털 시스템의 입출력 장치와
	그 장치에 표현된 내용
=> 외우고 예를 들어 쉬운 말로 설명해 봅시다.

UI 디자인
- 디지털 시스템의 입출력 장치의 내용과 모양을 디자인
	메뉴바, 버튼, 음성인식 장치 등


UX/UI 차이

UX Design (Process)
- 서비스가 아닌 사용자 관점에서
	새로운 경험을 제시하는 프로젝트 수행 (절차)
UI Design
- 서비스가 구제척으로 보여지는 형태, 정보의 배치와 구성


UX/UI 디자인 영역
- 일반적으로 IA, 이용흐름, 인터랙션, UI, GUI 등을 포함
순서

비주얼 디자인(GUI)
사용자 인터페이스(UI)
인터랙션 디자인(IXD)
정보 구조 및 흐름(IA, User Flow)
서비스(Content, Function)
전략 및 콘셉트(Concept)


UX/UI 디자인 영역 UI 디자인 영역
- UI 에서 interaction을 통해 UX 창출
	최신 UI < 정보를 잘 탐색하고 최적의 UX를 창출하는 UI
	UI는 경험을 디자인하여 사용자에게 전달하는 방법
1. 정보 구성 : 어떤 정보를 제공할지 정의, 연관관계 설계
2. 배치 : 시각적 계층구조 고려
3. 정보 제공의 형태 결정


IA(information architecture)
- 데이터를 수집하고, 이를 적절하게 정리해
	사용자에게 의미 있는 형태로 구성하여
	정보의 의미를 명확하게 전달하기 위해 구조화
	정보의 전체적인 체계, 논리적 구조, 검색 방법,
	필터나 태그, 레이블링 등 포함

이용 흐름(user flow)
- 사용자가 서비스를 이용하는 흐름
	사용자의 연속된 행동이
	실제 서비스에서 어떻게 흘러가는지를
	업무, 화면 단위로 정의

상호작용(interaction)
- 입출력 장치(Ui)를 매개로 디지털 시스템과 사용자가 주고 받는 일련의 릐사 소통

인터랙션 디자인
- 사용자의 행동과 이에 반응하는 시스템의 절차를 설계

GUI(graphic user interface)
비주얼 디자인
- 서비스/브랜드의 정체성, 개성, 고유한 심미성 표현, 정보의 가치를 높이는 데에 기여
- 사용자가 정보를 좀 더 잘 이해하고 사용할 수 있게 대비, 균형, 항조, 리듬 등의 조형적인 면을 고려하여 반영
	예) 서로 다른 정보들의 대비, 동질 정보의 약한 대비, 정보 내 특정 요소의 강조와 주목성
- 최신의 트렌드와 창의적인 감성도 중요하지만
	정보의 인식, 이해, 행동에 큰 영향을 미치기 때문에
	UX 중요하게 고려해야 한다.


UX/UI 디자인 영역

서비스
- 사용자에게 어떤 가치를 전달하기 위해서
	구체적인 정보/기능을 하나로 묶어서 제공하는 것
@서비스 경험 디자인
- 서비스를 사용하는 사람과 서비스를 제공하는 사람들에게
	최적의 경험을 주기 위한 원리와 방법을 설계하는 것.


UX 방법론 = 문제 해결 p.16
- 사용자의 문제를 찾아서 그 해결 방법을
	아이디어, 시나리오, 프로토타입 순서로 풀어나가는 것

디자인 씽킹
- 사용자 입장에서 그들의 경험을 깊이 공감하여
	무엇이 필요한지 찾고,
	통합적인 사고를 통해
	만족스러운 서비스를 디자인하는 패러다임
